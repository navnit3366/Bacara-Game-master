# Bacara Game

![BACARA](assets/game.png)


## Getting Started

1. Clone:
    ```bash
    $ git clone https://github.com/FelixLuciano/bacara-game.git
    ```
2. Enter the folder:
    ```bash
    $ cd Bacara-Game
    ```
3. Then run the game:
    ```bash
    $ python src\index.py
    ```
4. **Have fun!**
